# Google-Map-Distance-Calculator
This project calculates the distance between two geographical locations using latitude and longitude values. It also estimates the travel time based on the selected mode of transport.

## Description
The project uses the Haversine formula to calculate the distance between two points on the Earth. Based on the selected travel mode (walk, bike, or car), the total travel time is calculated and displayed in the browser console.

## Features
- Distance calculation using latitude and longitude
- Uses Haversine Formula
- Travel modes:
  - Walking
  - Bike
  - Car
- Time calculation in minutes or hours
- Console-based output

## Technologies Used
- HTML
- JavaScript

## Learning Outcome
- Understanding of Haversine formula
- JavaScript math functions
- Working with functions and variables
- Basic problem solving using JavaScript
